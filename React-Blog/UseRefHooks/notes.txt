1. controlled component
2. uncontrolled component

Jab bhi aap controlled and uncontrolled ke bare mein suno toh aap samajh lena ki
apke pass jo component hai uske andar input feilds hai ya phir ek form bana hai

controlled component ke andar kya hota hai en input feilds ki value state ke andar 
control kiya jata hai 

but uncontrolled component mein state ka use nahi kiya jata hai yahan koi state ka 
role nahi hota hai aapko directly dom ki help se dom object ki help se input value
ko get kiya jaega 

jab bhi ek input feild ko useRef se get karte ho usko kaha jata hai uncontrolled component par aisa nahi hai jab bhi kisi bhi input feild ki value get kar rahe ho 
aur wahan state ka use nahi hota hai and directly dom ka object use hota hai
